# MDX and JSX Integration

Learn how to use MDX with JSX in your Hono applications.

## What is MDX?

MDX allows you to use JSX in your markdown content.

## Benefits

- Write markdown with JSX components
- Create interactive documentation
- Build dynamic blog posts

## Example

```jsx
import { Button } from './components'

# Welcome

<Button>Click me!</Button>
```
